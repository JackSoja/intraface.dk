== Setup for Programming Intraface ==

Basic setup:

- Eqlipse with feature Subclipse
- Setup Ecplipse to replace tab with space

Server: 

- Apache server
- Mysql Server
- PEAR 
- Install dependencies from Intraface package.
- PHP module openSSL (for https requests to onlinepayment gateways)
- PHP with magic_quotes_pgc = Off

Unittesting:

- phpunit http://www.phpunit.de
- Xdebug kræver php source kode (php-devel)

Selenium testing:

- Selenium plugin for Firefox

Phing

- Phing
- Phing_IlibPearDeployerTask from public.intraface.dk
- If ftpDeployTask is not located in phing/tasks/ext/FtpDeployTask.php get it from 
  http://phing.info/trac/browser/branches/2.3/classes/phing/tasks/ext/FtpDeployTask.php
